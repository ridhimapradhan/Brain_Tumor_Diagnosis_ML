# Brain Tumor Diagnosis using Machine Learning

## Overview
This project detects brain tumors from MRI images using Machine Learning and Deep Learning techniques.

## Features
- Image preprocessing using OpenCV
- CNN model for classification
- Tumor vs Non-Tumor detection

## Technologies Used
- Python
- TensorFlow
- OpenCV
- NumPy

## Dataset
https://www.kaggle.com/datasets/navoneel/brain-mri-images-for-brain-tumor-detection

## Model Architecture
The project uses a Convolutional Neural Network (CNN) for image classification. 
The model consists of multiple convolutional layers, pooling layers, and fully connected layers to extract features and classify MRI images.

## Workflow
1. Load MRI image dataset
2. Preprocess images (resize, normalize)
3. Train CNN model
4. Evaluate model performance
5. Predict tumor presence

## Results
The model is expected to achieve high accuracy in classifying MRI images as tumor or non-tumor.

## Future Improvements
- Improve accuracy using advanced deep learning models
- Deploy as a web application
- Add multi-class tumor classification
